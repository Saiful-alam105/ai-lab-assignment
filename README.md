# AI Lab Assignment — KNN Algorithm

This repository contains the implementation of the K-Nearest Neighbor (KNN) algorithm from scratch.

## Assignment Tasks

### 1. KNN Classification
Dataset: Iris Dataset

Tasks:
- Dataset splitting
- Euclidean distance calculation
- KNN classification
- Validation accuracy calculation
- Best K selection
- Final test accuracy

### 2. KNN Regression
Dataset: Diabetes Dataset

Tasks:
- Dataset splitting
- Euclidean distance calculation
- KNN regression
- Mean Squared Error calculation
- Best K selection
- Final test MSE

---

## Technologies Used

- Python
- Pandas (only for dataset loading)
- Jupyter Notebook
